# Sketchfab API Community Docs

> ⚠️ **Unofficial, community-maintained documentation**  
> This repository is NOT an official Sketchfab publication. It is maintained by the community and may be incomplete or out of date. For official information, visit https://sketchfab.com/developers.

Reorganized, modular documentation for Sketchfab APIs (Data API v3, Download API, Viewer API, OAuth, oEmbed) with standalone examples. All new docs live under `docs/`.

## Getting started

1. Read `docs/00-introduction.md` for scope and navigation.
2. Check `docs/02-authentication.md` for token formats and OAuth flows.
3. Explore the API sections:
   - Data API: `docs/data-api/10-data-api-overview.md`
   - Download API: `docs/download-api/20-download-api-overview.md`
   - Viewer API: `docs/viewer-api/30-viewer-api-overview.md`
   - oEmbed: `docs/oembed/40-oembed-overview.md`
   - Guidelines: `docs/guidelines/50-developer-guidelines-overview.md`
4. Browse code samples in `examples/` (JavaScript, Python, curl).
